✓ Sudden death logic: only enters if scores tied AND sudden death enabled
✓ Break goal handling: confirms with user, handles tiebreak scenarios
✓ Timer system with proper period progression
✓ All game logic fully functional as described

GAME LOGIC FLOW:
===============
1. Start First Game At This Time → First Half → Half Time → Second Half
2. If scores tied at end AND overtime enabled: → Overtime periods
3. If scores still tied after overtime AND sudden death enabled: → Sudden Death
4. After Between Game Break: loop back to First Half (unless overtime/sudden death)
5. Special handling for goals added during breaks (can trigger overtime/sudden death)
